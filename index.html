AdminIsMaster Plugin

Ultimate Admin Control for Your Minecraft Server

Features

  • Admin Chat Channel: Create a private chat channel for admins using /adminchat.
  • Admin Teleportation: Teleport to players or teleport players to you with /teleport <player> and /teleporthere <player>.
  • Player Tracking: Track players' coordinates and status using /track <player>.
  • Instant Broadcast: Send server-wide messages with /broadcast <message>.
  • Admin Mode: Toggle admin mode for invisibility and invulnerability using /adminmode.
  • Automatic OP: Listed players get OP automatically upon joining the server.

Installation

  1. Download the AdminIsMaster plugin JAR file.
  2. Place the JAR file in the plugins folder of your Minecraft server.
  3. Restart or reload your server.
  4. Configure the plugin as needed by editing the plugin.yml file and setting permissions.

Usage

Once installed, the plugin allows you to manage server admins with ease. Use the provided commands to control admin chat, teleportation, tracking, broadcasting, and admin mode. Players listed in the configuration will receive OP automatically upon joining the server.
