AdminMaster Plugin

Welcome to the AdminMaster plugin! This plugin is designed to help server admins manage their Minecraft servers efficiently with a suite of powerful tools and commands.

Features

  • Admin Chat Channel: A private chat channel for admins.
  • Admin Teleportation: Teleport to any player or teleport players to you.
  • Player Tracking: Get live updates on a player's coordinates and status.
  • Instant Broadcast: Send server-wide messages in a special format.
  • Admin Mode: Toggle a mode where admins become invisible, invulnerable, and can fly.
  • Essential tools for large and small servers.

Installation

To install the AdminMaster plugin, simply download the .jar file and place it in your server's plugins folder.
