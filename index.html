AdminMaster Plugin

AdminMaster is a powerful Minecraft plugin designed to enhance server management for both small and large servers. It provides a variety of tools that allow admins to manage players, teleport, broadcast messages, track players, and much more.

Features:

  • Admin Chat Channel
  • Admin Teleportation
  • Player Tracking
  • Instant Broadcast
  • Admin Mode
  • Admin Clear Chat
  • Admin Heal
  • Admin Freeze

Installation:

To install the AdminMaster plugin, download the latest version from our GitHub releases page.
